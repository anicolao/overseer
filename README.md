# Overseer

Overseer is a GitHub-native agent coordination system designed to enable AI agents to collaborate autonomously.

## Overview

Overseer leverages GitHub Actions, Issues, and Pull Requests to orchestrate a team of LLM-based agents. Unlike its predecessor, Morpheum, which relied on Matrix for communication, Overseer is fully integrated into the GitHub ecosystem.

## Key Features

- **GitHub-Native:** Communication and execution happen directly on GitHub.
- **Autonomous Collaboration:** Specialized agents work together to deliver well-tested code.
- **Human-on-the-Loop:** Humans provide guidance and approval when needed.
- **Scaleable Architecture:** Designed to manage complex tasks across multiple repositories.
